About NewsHub

NewsHub tackles information overload by delivering curated news from diverse sources, inspired by platforms like Flipboard.

Key Features

  • Personalized feeds via multiple APIs (BBC, TechCrunch, Guardian, GNews, Mediastack).
  • User profiles and article saving with localStorage.
  • Accessible, responsive design with Material Design principles.
  • Advanced performance with lazy loading and debouncing.

Technical Approach

Built with vanilla HTML, CSS, and JavaScript, leveraging Fetch API, Intersection Observer, and Web Share API for a scalable, modern web app.
